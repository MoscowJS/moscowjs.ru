Инженерные данные в Web: Autodesk View and Data API
Владимир Ананьев

Предоставление доступа через Web к данным CAD-систем – геометрии и свойствам сложно структурированных моделей – всегда было весьма непростой задачей.

Мы представим Autodesk View and Data web service – одно из решений, которое претендует на самое широкое применение в этой области. Его REST API предназначен для выгрузки и подготовки к стримингу моделей самых разных исходных форматов (60+). JavaScript API предоставляет инструментарий для организации интерактивного просмотра 2D- и 3D-моделей и их инженерных данных непосредственно в web браузере без необходимости привлекать «родные» тяжеловесные CAD.
